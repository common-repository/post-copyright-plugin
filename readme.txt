== Description ==

This plugin simply adds a copyright notice to the bottom of every post on
your WordPress site.

Example: (c) Copyright (the year the post was made), (the author), All
Rights Reserved. Written For: (the name of your blog)

More features, like custom text, coming soon.
== Installation ==

1. Upload `post-copyright-plugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
